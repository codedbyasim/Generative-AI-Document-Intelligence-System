Project Information

Project Overview

Gen AI Document Intelligence is a full-stack AI-powered web application that extracts structured information and summaries from unstructured documents like scanned PDFs and images.

Core Functionalities

  • Upload scanned PDFs or images via drag-and-drop
  • Extract raw text using Tesseract OCR
  • Detect Named Entities with HuggingFace Transformers
  • Generate summaries using LLMs
  • Download results in JSON or PDF format

Technologies Used

  • Python + Flask (Backend Web Framework)
  • HTML5, CSS3, JavaScript (Frontend)
  • OpenCV & Tesseract OCR for text extraction
  • HuggingFace Transformers for NLP tasks
  • Bootstrap-style responsive design + animations

Future Enhancements

  • Table detection and structured parsing (AI Vision Models)
  • Auto classification of document types (Invoices, Receipts, etc.)
  • Dashboard with file history and visual analytics
  • User login system with upload history
